# 跨电脑使用说明

## 1. 使用方式

把整个文件夹复制到其他 Windows 电脑：

```text
工艺表格自动优化工具
```

进入文件夹后，双击：

```text
工艺表格自动优化工具.exe
```

即可打开可视化工具。

## 2. 不需要安装的内容

便携版已经包含运行所需环境。

其他电脑上不需要安装：

```text
Python
openpyxl
PyInstaller
```

## 3. 不要只复制 exe

请复制整个文件夹，不要只复制 `工艺表格自动优化工具.exe`。

因为程序旁边还需要：

```text
_internal
optimize_rules.json
使用说明书.md
项目蓝图.md
工艺表格自动优化模板.xlsx
```

## 4. 修改触发关键词

编辑同目录下：

```text
optimize_rules.json
```

或在可视化工具窗口中直接编辑关键词并保存。

## 5. Windows 安全提示

第一次运行时，Windows 可能提示“Windows 已保护你的电脑”。

如果确认来源是你自己的工具，可以点击：

```text
更多信息
仍要运行
```

## 6. 建议放置位置

建议放在普通用户目录，例如：

```text
D:\工艺表格自动优化工具
桌面\工艺表格自动优化工具
文档\工艺表格自动优化工具
```

不建议放在系统目录，例如：

```text
C:\Program Files
```

避免没有写入配置文件的权限。
