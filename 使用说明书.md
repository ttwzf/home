# 工艺表格自动优化工具使用说明书

## 1. 工具用途

本工具用于自动优化工艺 Excel 表格中的材料名称。

当某一行材料名称满足触发关键词，并且后面连续行满足规则时，工具会把后续连续行的材料信息补充到第一行材料名称后面的括号中。

示例：

```text
麂皮绒（有弹力）-需复合
0.85CM厚37#海绵（复合）  灰色
```

优化后：

```text
麂皮绒（有弹力）-需复合(0.85CM厚37#海绵*灰色)
```

## 2. 当前处理规则

工具会从表格中自动识别这些表头：

```text
材料名称
颜色
单位
单件用料
```

当第一行材料名称包含配置文件中的触发关键词时，工具会向下查找连续行。

只有后续连续行同时满足以下条件，才会被追加到第一行括号中：

```text
单位与第一行相同
单件用料与第一行相同
行号连续，中间没有断行
材料名称非空
```

追加格式为：

```text
后续行材料名称*后续行颜色
```

如果有多行连续满足条件，用 `+` 连接：

```text
第一行材料名称(下一行材料名称*颜色+再下一行材料名称*颜色)
```

后续行材料名称中的 `（复合）` 会被去掉后再追加。

## 3. 打开可视化工具

双击项目目录中的这个文件：

```text
打开可视化工艺表格优化工具.bat
```

打开后可以在窗口中完成以下操作：

```text
选择需要优化的 Excel 文件
设置输出文件路径
编辑触发关键词
选择是否标绿并添加批注
选择是否生成订单用量汇总表
选择是否只汇总连续同用量材料组
选择汇总表是否显示来源行号
点击“开始优化”
查看本次优化结果
```

## 4. 使用步骤

1. 双击 `打开可视化工艺表格优化工具.bat`。
2. 点击“选择”，选择需要处理的 `.xlsx` 文件。
3. 确认输出文件路径，默认会生成：

```text
原文件名_已优化.xlsx
```

4. 检查触发关键词。
5. 点击“开始优化”。
6. 完成后打开生成的 Excel 文件检查结果。

## 5. 触发关键词配置

触发关键词保存在：

```text
optimize_rules.json
```

默认内容：

```json
{
  "trigger_keywords": [
    "需复合"
  ]
}
```

表示只有第一行材料名称中包含 `需复合` 时，才会触发优化。

如果需要增加多个关键词，可以改成：

```json
{
  "trigger_keywords": [
    "需复合",
    "里衬",
    "复合海绵"
  ]
}
```

如果希望任意非空材料名称都可以作为第一行触发，可以改成：

```json
{
  "trigger_keywords": []
}
```

也可以直接在可视化工具窗口中编辑关键词，一行一个，然后点击“保存关键词”。

## 6. 输出结果说明

工具默认不会覆盖原文件，而是另存为新的 Excel 文件。

被自动优化的第一行材料名称会：

```text
标浅绿色
添加批注
保留原有行
保留原有公式
保留后续连续行
```

订单用量列不会被重新计算，原公式会继续保留。

## 7. 订单用量汇总表

可视化工具默认会在输出 Excel 中新增一个工作表：

```text
订单用量汇总
```

汇总表按照以下字段分组合计订单用量：

```text
材料名称
颜色
单位
门幅/规格
```

其中 `材料名称` 使用优化后的材料名称。

默认情况下，可视化工具会勾选：

```text
只汇总连续同用量材料组
```

此时汇总表只统计参与连续材料组的行。连续材料组需要满足：

```text
第一行材料名称命中触发关键词
后续行与第一行单位相同
后续行与第一行单件用料相同
行号连续
```

如果取消这个选项，汇总表会改为全表汇总。

汇总表默认列为：

```text
材料名称
颜色
单位
门幅/规格
订单用量合计
来源行号
```

`来源行号` 是可选项，可以在可视化工具中取消勾选。

汇总时会排除以下行：

```text
材料名称为空
颜色为空
单位为空
订单用量为空
订单用量为 0
```

如果订单用量是类似 `=F6*$G$5` 的简单公式，工具会按公式引用算出结果后再汇总。

## 8. 常见情况判断

### 8.1 为什么相邻两行没有合并到括号里？

请检查以下条件：

```text
第一行材料名称是否包含触发关键词
两行单位是否相同
两行单件用料是否相同
材料名称是否为空
是否中间有断行
```

其中 `单位不同` 或 `单件用料不同` 时，不会追加。

### 8.2 为什么已经优化过的文件再次运行没有变化？

工具有防重复机制。

如果第一行已经包含相同的括号内容，再次运行不会重复追加。

### 8.3 能不能不标绿、不加批注？

可视化工具中取消勾选：

```text
标绿并添加批注
```

即可只修改材料名称文字。

### 8.4 能不能不生成汇总表？

可以。

在可视化工具中取消勾选：

```text
生成订单用量汇总表
```

### 8.5 能不能汇总全表？

可以。

在可视化工具中取消勾选：

```text
只汇总连续同用量材料组
```

取消后会按 `材料名称 + 颜色 + 单位 + 门幅/规格` 汇总所有有效材料行。
